# 洛塞娅 — 游戏设计文档

## 1. 核心体验

四人六边形资源格策略桌游（卡坦式）：掷骰收割、修路建村升城、贸易与发展卡争夺 10 分胜利。首版以**阶段日志 + 手动「下一阶段」**推进，每步打印状态便于校验规则。

## 2. 游戏形态

| 项 | 说明 |
|---|---|
| 玩家 | 4 人（玩家 1 人类，2–4 简单 AI） |
| 地图 | 中心沙漠 + 三圈共 37 格六边形（外围每边 5 格） |
| 资源 | 木、砖、石、粮、羊 |
| 建筑配额 | 每人 5 道路、5 村庄、5 城镇 |
| 胜利 | 总分 ≥ 10 立即结束 |

## 3. 地图与数字

- 沙漠格居中，强盗初始在此；被占后该格永不产出
- 资源数字按附录分配（草原 2/10/11、麦田 3/12/4、矿山 6/9、砖厂 5/8、森林 6/7）
- 六条海边各 2 个港口顶点（实际每处仅 1 村）：3:1×2、2木:1×1、2石:1×1、2砖:1×1

## 4. 关键模块

| 模块 | 职责 |
|---|---|
| `map` | 六边形拓扑、顶点/边 ID、港口 |
| `rules` | 建造合法性、收割、强盗、弃牌、VP |
| `phase-controller` | 开局/setup/回合阶段机 |
| `ai` | 非人类自动落子 |
| `board-ui` | SVG 六边形棋盘 |
| `hud` | 日志、资源、阶段按钮 |

## 5. 分阶段计划

### 阶段 1（当前）
- 地图生成 + 完整规则引擎
- 开局：轮流空降 1 村 + 2 道路
- 首回合：掷骰→收割/强盗→自由发展（日志打印）
- BGM + 点击/掷骰音效

### 阶段 2
- 人类点击选点建造
- 发展卡完整交互、玩家间交易
- 成就分（最长路、最多骑士）实时争夺

### 阶段 3
- 3D 场景 pack 与 DOM 棋盘同步高亮
- 动画与本地多人
